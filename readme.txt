== Description ==

The "Redirect After Comment Per Page" plugin will redirect automatically the commenter to a specific page after they validate a comment on your blog. Each page can have a different redirection.

How to use :
- In admin panel, look in the section "Settings / Comment After Redirect" and
  enable plugin
- When you edit a page, just fill the redirect URL below the page content
  textarea

Why would you use that plugin ?
- To offer a specific gift when they put a comment (during a product launch for example)
- To forward to a survey
- To call to share on social networks

More info on : http://www.anyideas.net/redirect-after-comment-per-page-wordpress-plugin/

== Installation ==

1. Upload `redirect-after-comment-per-page' folder to the `/wp-content/plugins/` directory.
1. Activate the plugin through the 'Plugins' menu in WordPress.
1. Configure settings in control panel.
