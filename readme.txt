Redirects commenters to a predefined URL after comment submission. Each page/post can have a different redirection.

== Description ==

The "Redirect After Comment Per Page" plugin will redirect automatically the commenter to a specific page after they validate a comment on your blog. Each page can have a different redirection.

Useful to engage visitors during a product launch for example.

* More info on : http://www.anyideas.net/redirect-after-comment-per-page-wordpress-plugin/
* Plus d'info sur : http://www.anyideas.net/redirect-after-comment-per-page-wordpress-plugin-francais/

== Installation ==

1. Upload 'redirect-after-comment-per-page' folder to the '/wp-content/plugins/' directory.
1. Activate the plugin through the 'Plugins' menu in WordPress.
1. Configure settings in control panel.

* More info on : http://www.anyideas.net/redirect-after-comment-per-page-wordpress-plugin/

== Frequently Asked Questions ==

= Is it working with redirection to external websites ? =

Unfortunately no. You need to use this plugin with another one called "Quick
Redirects" to redirect to an external URL.
